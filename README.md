# 遗传系谱图学习工具（高中版）

一个面向高中生物教学的交互式遗传系谱图绘制与概率计算工具，支持四种经典孟德尔遗传模式。

## ✨ 功能特点

- **四种遗传模式**：常染色体隐性（AR）、常染色体显性（AD）、X 连锁隐性（XLR）、X 连锁显性（XLD）
- **可视化系谱图**：方形（男）/ 圆形（女），实心表示患病，紫色标注携带者
- **交互式操作**：点击节点快速编辑，支持缩放拖拽视口
- **基因型管理**：根据当前遗传模式自动切换可选基因型
- **子代概率计算**：选择父母后自动计算患病、携带、正常的概率
- **数据导出/导入**：JSON 格式保存与加载，方便教学复用
- **深浅色主题**：自动跟随系统或手动切换
- **纯前端运行**：无需服务器，浏览器打开即可使用

## 🛠️ 技术栈

- HTML5 + CSS3（自定义属性、响应式布局）
- JavaScript（ES6，Canvas SVG 绑定）
- SVG 绘图 + 事件驱动的系谱布局算法
- 孟德尔分离定律计算引擎

🎓 教学使用建议

    在教师机上打开工具，选择遗传模式（如 AR）。

    添加家族成员，设定患病状态和基因型。

    使用“子代遗传概率计算”模块，选择父母后实时展示子代患病概率。

    导出 JSON 文件，分享给学生练习。

⚙️ 自定义

    修改 index.html 中的示例数据或默认模式。

    调整 style.css 中的 :root 变量可改变主题配色。

    概率计算逻辑集中在 script.js 的 computeMendel 函数中，可根据教材调整。
